## About This Project

A dark minimal portfolio site built with Astro 5 and deployed to Cloudflare Pages for free.

### Features

- View Transitions API for smooth page navigation
- Scroll-driven animations using pure CSS
- CSS nesting, `@layer`, `color-mix()` and modern selectors
- Cloudflare Access with Google OAuth for private content protection
- Zero JavaScript shipped by default (Astro islands architecture)

### Tech Stack

- Astro 5 (hybrid rendering)
- Vanilla CSS with modern features
- Cloudflare Pages (hosting)
- Cloudflare Access (auth)
